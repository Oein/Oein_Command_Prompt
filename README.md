# Oein Command Prompt
### 
## Commands List
### echo [ data ]
#### ex : echo Hello?
### bash [ File Name ]
#### ex : bash a.obash
### title [ Title ]
#### ex : title OCP
### dir
#### ex : dir
### ls
#### ex : ls
### timeout [ millisecond ]
#### ex : timeout 1000
### sleep [ millisecond ]
#### ex : sleep 1000
### mkdir [ Dir name ]
#### ex : mkdir A_New_Folder
### rmdir [ Dir name ]
#### ex : rmdir An_Older_Folder
### cd [ Dir name ]
#### ex : cd A_Folder
### set [Key]=[Value]
#### ex : set a_var=32
### playsound [wav file name]
#### ex : plausound A_Sample_Wav_File.wav
### stopsound
#### ex : stopsound
### beep [ octave + note in English ] [ millisecond ]
#### ex : 4C 400
#### Octave : 1C ~ 8B
### help
#### ex : help
### ver
#### ex : ver
### VFile [File Name]
#### ex : VFile A_Text_File.txt
